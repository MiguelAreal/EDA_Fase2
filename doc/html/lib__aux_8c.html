int  tamanho

Adiciona um novo caminho à estrutura de Caminhos.

Esta função cria um novo nó de caminho, copia os vértices do caminho fornecido para o novo nó, e então adiciona este nó à lista ligada de caminhos. A lista de caminhos é atualizada para incluir o novo caminho no início da lista.

Parâmetros
listaPonteiro para o ponteiro da lista de caminhos. É usado para atualizar a lista com o novo caminho.
caminhoArray de inteiros representando a sequência de vértices no novo caminho.
tamanhoO número de vértices no array do caminho.
Retorna
int 0: Sucesso.

◆ print_grafo()

void print_grafo ( GR *  grafo)

Exibe na consola uma visualização do grafo, mostrando as conexões entre vértices e o peso de cada aresta.

Esta função percorre todos os vértices do grafo e imprime informações sobre as arestas conectadas a cada vértice. Cada linha começa com o índice do vértice, seguido por uma lista de todas as arestas conectadas a ele, incluindo o destino da aresta e o peso associado. Os vértices são numerados de 1 a n, onde n é o número total de vértices no grafo.

Parâmetros
grafoInstância do grafo.
