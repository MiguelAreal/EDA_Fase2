\doxysubsection{Descrição detalhada}
Estrutura para armazenar caminhos com a maior soma. 

Esta estrutura é usada para armazenar uma lista ligada de caminhos no grafo. Cada caminho é representado por uma sequência de vértices e o tamanho dessa sequência. A estrutura também possui um ponteiro para o próximo caminho na lista, permitindo a criação de uma lista ligada de caminhos.
